Automatisation d'un Sudoku en C

Méthodes du Sudoku :

En équipe de 2, nous avons créer un premier algorithme remplissant les grilles de Sudoku selon les méthodes suivantes :

- Singleton nu
- Singleton caché
- Paires nues
- Paires cachées

Ce sont des méthodes rapide et efficaces sur des grilles simples et basiques

Méthode du backtracking :

Cette méthode nous a permis de rempir n'importe quelle grille de Sudoku (solvable) c'est une méthode efficace mais très lente.

Code source définitf :

Nous avons développés un programme permettant de remplir des grilles en format 4x4, il se base sur les méthodes de singleton et de paires et termine la complétion de la grille avec du backtracking. C'est un programme très rapide pour des grilles simples et plutôt lent pour des grilles avancées.

Optimisation du site web PACT

Objectif :

Réalisation d'un site web pour une entreprise fictive de tourisme nommée PACT. l'Objectif étais de concevoir un site web vitrine pour cette entreprise et de mettre en avant des destinations et des bons plans.

Contrainte :

Le site web est assez volumineux et peut être lent si mal optimisé, de plus, en cas de forte afluence, si de nombreux visiteurs ou professionnels interagissent avec le site, cela peut ralentir le site. Le but étais donc d'optimiser le projet dans le but d'améliorer ses performances en toutes situations.

Optimisation :

Nous nous sommes donc concentrés sur l'optimisation de notre code dans le développement du projet, nous avons fais en sortes d'optimiser au maximum nos fichiers et avons fais en sortes d'alléger les parties et les opérations les plus lourdes sur le site.

Résultat :

Cette optimisation nous donne un site web fluide et plus léger, agréable à utiliser, également dans des moments de fortes affluence.
