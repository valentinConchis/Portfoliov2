Réalisation d'un site de streaming

Expression des besoins :

Dans un premier temps, nous avons résumé l'interview du commanditaire du site, ce qui nous as permis de nous organiser pour développer correctement le site par la suite.

Conception de la charte graphique

Nous avons réalisés ensuite la charte graphique, charte qui définis les couleurs et le design que le site prendra à sa conception. Cette charte a été inspirée du site Netflix.

Création de la maquette du site web

Cette maquette a été le premier appercu du site web final, nous l'avons faite pour deux types d'écrans, le format PC et portable. C'est en se basant sur cette maquette que nous avons réalisés par la suite le site web.

Edification finale du site web

Enfin, en fin de projet nous avons réalisés le site internet en HTML, CSS et JavaScript. Certains aspects du rendu final diffèrent de la maquette, certaines améliorations ont étés apportées eaprès la création de la maquette.

Création d'un site web pour la PACT

Objectif :

Réalisation d'un site web pour une entreprise fictive de tourisme nommée PACT. l'Objectif étais de concevoir un site web vitrine pour cette entreprise et de mettre en avant des destinations et des bons plans.

Cahier des charges :

Le site devait être réalisé d'une part pour des visiteurs classiques n'ayants pas de comptes, pour les abonné ayant crées un compte et pour les professionnels ayant besoins d'une interface adaptés afin de pouvoir poster leurs activitées et afin de pouvoir observer les retours clients.

Maquettage :

Cette étape du projet a été nécessaire à la création du design du site, le maquettage nous a permis de définir le design et l'identitée visuelle du site avec le client (un jury composé d'environ 5 personnes).

Développement du site :

Le développement du site s'est réalisé dans une équipe de 7 personnes, les langages employés étaient le PHP, le JS, CSS, PosgreeSQL.
